# Friendship Calculator 💖

## About
Friendship Calculator is a simple web application built using Python and Streamlit. Users can enter two names, and the app generates a random friendship score between 90% and 100% with a fun celebration.

## Features
- Enter two names.
- Generates a random friendship score.
- Displays the result instantly.
- Balloon animation for a fun user experience.

## Technologies Used
- Python
- Streamlit
- Random module

## Installation

1. Clone the repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python -m streamlit run friendship.py
```
